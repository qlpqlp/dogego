Built beyond Dogecoin Core

DogeGo ships operator tooling, indexing, and modern network modes that Core never bundled, in one Go binary with a browser dashboard.

shield

Post-Quantum commitments

First Dogecoin full node with Phase-1 PQ OP_RETURN carriers (FLC1, DIL2, RCG4). Send with automatic wallet commitments, verify via dogego_verifypqcommitment, and inspect in the explorer.

compress

zstd block compression

Optional zstd on stored block bodies plus a bundled blk*.dat layout. Core keeps raw blocks uncompressed. DogeGo reports compression ratio and disk savings in the dashboard.

dns

Full node or SPV

Run headers-first IBD with full validation, or switch to lightweight SPV (headers only) for minimal disk. Same binary, same wizard. Pick your profile on first run.

satellite_alt

CGNAT & Starlink relay

P2P modes: classic, cgnat (outbound-only), or both. Up to 12 outbound peers relay blocks without inbound port forwarding, built for carrier NAT and Starlink.

science

Reboot testnet

Fresh testnet with addresses starting with T (for test). RelaxedPoW enables solo mining at low hashrate, so the chain does not stall when hash power drops. Port 44556.

search

Search everything

Full tx index by default. Search height, block hash, txid, or address from the dashboard. Dogecoin is a public ledger, and DogeGo makes every entry easy to find.

dashboard

Web dashboard

Loopback UI at :2013: KPI cards, mempool, analytics charts, settings, docs, and a five-step setup wizard. No Qt install required.

integration_instructions

180+ JSON-RPC methods

Core-compatible HTTP JSON-RPC with cookie auth, ZMQ notifications, OpenRPC at /api/openrpc.json, and live capability flags in the Features tab.

account_balance_wallet

Built-in HD wallet

Send, receive, and history with BIP44 HD wallet, PIN unlock, optional WebAuthn biometrics, and PQ commitment overrides on send.

BlockStep: explore Dogecoin the fun way

Because Dogecoin is a public ledger, every block and transaction should be easy to find. DogeGo indexes the chain and BlockStep turns it into an interactive timeline.

Honest scope

DogeGo is experimental and under active development toward Dogecoin Core parity. It is not certified for exchanges, mining pools, or production wallets. For established operator SLOs, full script corpus parity, or wallet.dat compatibility, use Dogecoin Core.
